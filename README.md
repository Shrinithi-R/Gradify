# Gradify

Gradify is a Career Growth & Placement Readiness Platform built using React and Vite.

## Features

* User Authentication
* Profile Management
* GitHub Progress Tracking
* LeetCode Progress Tracking
* Resume Analyzer
* Analytics Dashboard
* Placement Challenges
* Settings Management

## Tech Stack

* React.js
* Vite
* JavaScript
* HTML
* CSS
* Git & GitHub
